# p7-Kasa
Projet 7 - OpenClassrooms

## Description
Ce projet est une application web construite avec **React** et **Vite** pour gérer une plateforme de location de logements.

## Prérequis

Avant de commencer, assurez-vous que vous avez installé **Node.js** et **npm**. Vous pouvez vérifier si vous les avez installés en exécutant les commandes suivantes dans votre terminal :

```bash
node -v
npm -v

1.Clonez le dépôt : Clonez ce projet sur votre machine locale avec la commande suivante :
git clone https://github.com/votre-utilisateur/p7-Kasa.git

2.Installez les dépendances : Rendez-vous dans le dossier du projet et installez les dépendances nécessaires avec npm :
cd p7-Kasa
npm install

3.Démarrez le serveur de développement : Une fois les dépendances installées, vous pouvez démarrer le serveur de développement avec Vite :
npm run dev

L'application sera disponible à l'adresse http://localhost:5173.

Pour lancer l'application en mode production, vous pouvez utiliser les commandes suivantes :
npm run build
npm run serve

Technologies utilisées :
React : pour la création des composants interactifs
Vite : pour le bundling et le démarrage rapide
CSS : pour la mise en forme
React Router DOM : pour la gestion de la navigation
